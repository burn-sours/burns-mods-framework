# Variable: LaraHairLeftX

## Description
X position of Lara's left hair braid. Used for hair physics/rendering.

## Details

| Field     | Value           |
|-----------|-----------------|
| Type      | `Int32`         |

## Usage
### Calling from mod code
```javascript
const hairX = game.readVar(game.module, 'LaraHairLeftX');
```
